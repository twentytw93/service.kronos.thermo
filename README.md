# Kronos Thermo

Kronos Thermo is a program add-on for **Kodi 21.x (Omega)** designed for **LibreELEC** systems. 
It monitors device temperature in real time and enforces safety measures to protect the system from overheating.

---

## Features
- **Temperature Monitoring** – Constantly checks system thermal levels during use. 
- **Warning Alerts** – Displays a notification when the temperature reaches **85°C**. 
- **Critical Protection** – At **95°C**, all playback is stopped, a warning is shown, and the user is prompted to shut down the system. 
- **Fail-Safe Design** – Ensures monitoring runs only after Kodi UI is ready, with built-in abort safety. 
- **Lightweight Operation** – Minimal resource usage for stable performance on Kodi.

---

## Compatibility
- Kodi **21 Omega** 
- LibreELEC 12.0.2+ 
- Python 3 (bundled with Kodi)
